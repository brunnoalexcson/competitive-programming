# 🟡 Ad-Hoc Problems

This directory contains solutions to **Ad-Hoc** problems from the Beecrowd platform.

## 🤔 What is Ad-Hoc?

Ad-Hoc problems are those that:

* ❌ Do not follow a standard algorithm pattern
* ❌ Do not have a predefined formula or method
* ✅ Require direct logical reasoning
* ✅ Often involve simulation and creativity

## 🧠 Skills Developed

* Advanced logical thinking
* Careful interpretation of problem statements
* Building solutions from scratch
* Handling multiple rules and edge cases

## ⚠️ Challenges

This category can be deceptively difficult because:

* Problems may look simple at first glance
* Small details can completely change the outcome
* Edge cases are very common
* Clean and well-thought-out code is essential

## 🛠️ Common Approach

When solving Ad-Hoc problems:

1. Read the problem carefully (more than once if needed)
2. Manually simulate sample cases
3. Identify patterns or rules
4. Then implement the solution

Each file represents an individual solution.
