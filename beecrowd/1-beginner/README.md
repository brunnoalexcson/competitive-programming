# 🟢 Beginner Problems

This directory contains solutions to **Beginner-level** problems from the Beecrowd platform.

Despite the label, many of these problems go beyond the basics and require:

* 🧠 Solid logical reasoning
* 🔍 Careful problem interpretation
* 🧮 Accurate input/output handling

## 🎯 Purpose

The goal of this section is to strengthen core programming fundamentals, including:

* Conditional statements (`if`, `else`)
* Loops (`for`, `while`)
* Basic mathematics
* Input and output handling
* Code organization

## ⚠️ Note

Even though these are classified as "Beginner" problems, many of them:

* Include tricky edge cases
* Require more refined logical thinking

## 🛠️ Technologies

* Language: C / C++
* Compiled via IDE (Vscode)

## 📌 Structure

Each file corresponds to a specific problem solution.

1001_problem-name.c
